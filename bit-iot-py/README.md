# Python으로 AWS IoT 사용하기

## 프로젝트 준비
  - python 설치 확인
    - python --version
    - python3 --version

- python ssl 버전 확인
    - python
      - import ssl
      - ssl.OPENSSL_VERSION
  - python3 ssl 버전 확인
    - python3
      - import ssl
      - ssl.OPENSSL_VERSION
- AWS IoT 파이썬 모듈 설치
  - python
    - pip install AWSIoTPythonSDK
  - python3
      - pip3 install AWSIoTPythonSDK
